Testing Process: Testers were given 2 minutes to test the game before being able to communicate with them.

Tester 1

During testing:

  • Turned up volume a lot on first page after selecting music (I had to whisper to them "it might be loud" in fear that the music would blast once the game started...)
  • Said "so cute" multiple times

Thoughts after testing:

  • "Really shows you put a lot of work in it
  • "Amazing how you were able to stick with the theme of Animal Crossing [through code and design]"
  • Timmy and Tommy placeholders for player name inputs added to the Animal Crossing theme and was a good indicator for what to write in the field (as opposed to just putting "Player 1" and "Player 2")
  • Background music is super cute
  • Could make a pop up like a text bubble that says "rules" when the user hovers over the info button so that they know it's for the rules
  • Hard to tell which face the dice is supposed to be read

Tester 2

During testing:

  • Spent a long time on rules page

Thoughts after testing:

  • Suggested there be a built-in legend key on the game interface to keep track of how the scoring works

Tester 3

During testing:

  • There was a single die on the screen at some point, not really sure how that happened (supposed to be four dice at all times, maybe a glitch?)
  • Pass button seems like it has confused all three testers

Thoughts after testing:

  • "I really like the visuals, they're very in tune with your Animal Crossing inspo and the little character animations are so cute, especially the speech bubbles"
  • "I wish the buttons were stylized to fit the Animal Crossing aesthetic more because everything else looks so cohesive and nice"
  • "It'd be nice if there was a field that showed you how much you rolled"
  • Put characters from the home page on the game interface
  • "Your dice are beautiful"
  • "The user flow was very clear"
  • "I wish there was a quit button on the game screen"

Conclusion: I got a lot of great feedback for the last iteration of my final project.

  • More clearly highlighting whose turn it is

  • Making the rules more readable / easier to understand
  • Re-skin dice to make them easier to read
