Overlays

 October 20th, 2021

In “In “Best Practices for Modals / Overlays / Dialog Windows” by Naema Baskanderi, the author highlights 6 key practices for using dialog windows. These practices include: giving the user an escape hatch, give the window a descriptive title, button labels should have clear names with an action, the size of the window should not exceed 50% and should be located in the upper half of the screen, the window should be highlighted compared to the background for focus, and finally the window should be user initiated — not a surprise pop-up.

Overall, all of these ideas are extremely useful points to keep in mind to create a seamless dialog window. In the future, if I must integrate modals / overlays / dialog windows, I will definitely refer back to this article to make sure I am on the right track.
